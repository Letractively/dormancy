public T findPendant (T obj, Collection<?> collection)

Iterates over the given collection, looking for an object which is semantically equal to a certain object.
An object is semantically equal if one of the following conditions apply:

  • the given object overrides the equals(Object) equals methods, which returns true for another object
  • the given object has a non-null identifier and its type as well as the identifier value is equals to the type and identifier value of another object

public abstract PropertyAccessor getPropertyAccessor (PMD metadata, Object obj)

Returns a PropertyAccessor for accessing the objects properties.

This method automatically detects the best strategy for accessing field values, which is either property access (by using org.springframework.beans.BeanWrapper BeanWrapper or field access (by using org.springframework.beans.DirectFieldAccessor DirectFieldAccessor.

The decision is made by applying the following algorithm:
If there is persistence metadata available and

  • if the javax.persistence.Id annotation was found on a getter method or
  • if the object is a proxy modified by Javassist and its but it is not accessible
the properties are access via getter and setter methods.
Otherwise the fields are accessed directly via reflection.

Note that if the given object is an uninitialized Javassist proxy, the object becomes initialized immediately if the direct field access strategy is chosen.
Otherwise Hibernate will trigger the initialization automatically upon getter invocation as usual.

Parameters
metadata the class metadata
obj the object
Returns
  • the property accessor

public abstract boolean isUninitialized (String propertyName, Object dbObj, Object dbValue, Object trValue)

Checks if a certain property of a JPA entity proxy is uninitialized and the corresponding value of the transient entity is null.
If the property of the persistent entity is uninitialized, it is assumed to be loaded lazily. Thus the property of the transient entity must be null. If is is not, an exception is thrown.
