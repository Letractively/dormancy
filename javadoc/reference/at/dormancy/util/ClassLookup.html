Class Overview

Performs class lookups using the default ClassLoader.

Note that none of the methods throw a ClassNotFoundException. Instead, null is returned unless something else is provided.

public static Class<T> forName (String name)

Returns the Class object associated with the class or interface with the given name.

If a ClassNotFoundException was thrown by the ClassLoader performing the lookup, it gets caught gracefully and null is returned.
Any other serious exception such as ExceptionInInitializerError or LinkageError is forwarded to the caller immediately.

Parameters
name the full-qualified name of the desired class.
Returns
  • the Class object for the class with the specified name or null.

public Class<T> get ()

Performs the lookup with the full-qualified class names until a Class was found.

If an exception was provided, it is thrown if no class was found. Otherwise, the provided default class or null is returned if none was set.

If the exception is checked i.e., it is neither of type RuntimeException nor Error, it gets wrapped in a RuntimeException.

Returns
  • the class found or null if the lookup failed and no default was provided

public List<Class<? extends T>> list ()

Performs the lookup with the full-qualified class names and returns all classes found.

If an exception was provided, it is thrown if no class was found.
The resulting list does neither null values nor the default class (if provided).

If the exception is checked i.e., it is neither of type RuntimeException nor Error, it gets wrapped in a RuntimeException.

Returns
  • the class found or null if the lookup failed and no default was provided
