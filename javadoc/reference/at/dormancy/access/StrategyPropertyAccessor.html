Class Overview

Uses the provided AbstractPropertyAccessStrategy to determine how to access an objectÂ´s properties.

If a property should be accessed, this implementation performs a lookup of the appropriate strategy for the requested property. Afterwards, another PropertyAccessor is used for accessing it.
If the property should be accessed via the getter and setter methods, a BeanWrapperImpl will be used.
Otherwise, a DirectFieldAccessor is used to access the properties directly without possible side-effects caused by invoking getter or setter methods.

public T convertIfNecessary (Object value, Class<T> requiredType, MethodParameter methodParam)

Casts the value to the required type.

This method can be overridden to enable the PropertyAccessor to perform advanced property conversion e.g., by using a org.springframework.core.convert.ConversionService ConversionService.

Parameters
value the value to convert
requiredType the type to convert to (or null if not known, for example in case of a collection element)
methodParam the method parameter that is the target of the conversion (for analysis of generic types; may be null)
Returns
  • the new value, possibly the result of type conversion
