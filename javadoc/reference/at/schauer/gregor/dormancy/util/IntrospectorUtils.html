Class Overview

Utility class that simplifies bean introspection and caches PropertyDescriptors for improved performance.

There are some alternative solutions e.g.,

  • Spring´s org.springframework.beans.CachedIntrospectionResults CachedIntrospectionResults that is a little bit slower but provides more features. Note that it is NOT intended for direct use by application code.
  • Apache´s org.apache.commons.beanutils.PropertyUtils PropertyUtils, which are comparable to CachedIntrospectionResults
  • Spring´s org.springframework.beans.ConfigurablePropertyAccessor ConfigurablePropertyAccessor and org.springframework.beans.BeanWrapper BeanWrapper do not use PropertyDescriptors but also provide a convenient way for accessing bean properties
