Class Overview

Simply returns the given entities.

This implementation can be used for disabling EntityPersisters for a certain method.

Example: 

 @PersistenceEndpoint
 public class OrderService {
 public OrderStatus order(Order arg) {
 // process order...
 return orderStatus;
 }
 

 @PersistenceEndpoint(type = NoOpEntityPersister.class)
 public String addCoupon(String code) {
 // process coupon code
 return message;
 }
 }
