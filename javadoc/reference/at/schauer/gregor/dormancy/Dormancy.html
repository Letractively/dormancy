public void addEntityPersister (AbstractEntityPersister entityPersister, Class... types)

Registers the given AbstractEntityPersister for certain types.
The AbstractEntityPersister is registered for every type returned by getSupportedTypes() and the parameter types. Furthermore, the type of the AbstractEntityPersister itself is registered so it can be used by in at.schauer.gregor.dormancy.interceptor.PersistenceEndpoint#types().

Parameters
entityPersister the EntityPersister to register
types the types of objects supported by the EntityPersister (may be null)
