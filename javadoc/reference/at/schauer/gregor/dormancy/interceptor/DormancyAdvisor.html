Class Overview

Intercepts calls and performs cloning and merging of Hibernate entities.

At first, an EntityPersister is resolved in the following sequence:

  1. Use the metadata of the PersistenceEndpoint annotation on the method to invoke
  2. Use the metadata of the PersistenceEndpoint annotation on the object instance
If any PersistenceEndpoint annotation is present, get an EntityPersister from the org.springframework.beans.factory.BeanFactory with the given name. Otherwise, get an EntityPersister of the specified type from Dormancy. If no appropriate EntityPersister is registered, Dormancy will be used.
